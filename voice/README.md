# Voice Templates

This folder provides a public-safe starting structure for an optional Dittonomics voice layer.

Use it only if you want your writing workflows to preserve a recognizable prose style across sessions. If you do not need that, you can ignore this folder entirely.

## Suggested Files

- `core_voice.template.md`
- `voice_examples.template.md`
- `econ_paper_register.template.md`
- `style_hierarchy.template.md`

Optional overlays:

- `collaboration_register.template.md`
- `journal_tightening_register.template.md`

## How To Use It

1. Copy the templates into your private `~/voice/` directory.
2. Rename them without the `.template` suffix.
3. Replace the prompts with your own prose preferences and examples.
4. Keep them private unless you explicitly want to publish your voice.

## Public-Safe Rule

The public Dittonomics repo documents the structure and precedence of a voice layer, but it should not publish one person's private writing system as the default voice for everyone else.
